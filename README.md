## Como ejecutar el proyecto

Primero instala las dependencias

```bash
npm install
```
Luego, corre el servidor
```bash
npm run dev
```
**Nota:** El servidor Node.js estará en funcionamiento en el puerto 3001 (http://localhost:3001)
